# VH Colliance Backend API

## Overview
Backend API for VH Colliance worker platform connecting clients with domestic and professional workers.

## Features
- 🔐 Authentication with JWT and OTP verification
- 👥 User roles: Client, Worker, Admin
- 📍 Real-time worker location tracking
- 💬 Real-time chat with Socket.io
- 💳 M-Pesa integration with 25% commission
- 📊 Admin dashboard with communication monitoring
- 🏆 Worker skills showcase and testimonials
- 📅 Booking management system

## Tech Stack
- Node.js & Express.js
- MongoDB with Mongoose
- Socket.io for real-time chat
- JWT for authentication
- M-Pesa Daraja API
- Cloudinary for file uploads
